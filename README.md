## Install

### Basic utilities

```bash
# bare minimums
curl -fsSL sh.mohitsingh.in/essential | bash
# dot configs
curl -fsSL sh.mohitsingh.in/shell | bash
```

### VSCodium

```bash
curl -fsSL sh.mohitsingh.in/code | bash
```

### NodeJS and Pyhton Globals

```bash
curl -fsSL sh.mohitsingh.in/globals | bash
```
